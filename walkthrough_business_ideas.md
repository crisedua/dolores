# Business Ideas Feature Walkthrough

This guide details how to verify the new "Business Ideas" feature.

## 1. Access the Feature
1. Open the application.
2. In the Sidebar, locate the **"Discovery"** section.
3. Click on the new **"Business Ideas"** (or "Ideas de Negocio") link with the "New" badge.

## 2. Onboarding Wizard ("Diseña tu Futuro")
1. Verify the page background is **White** (Light Theme).
2. **Step 1: Location**:
   - Select a country from the dropdown.
   - Click "Siguiente".
3. **Step 2: Situation**:
   - Verify the 3 options are displayed with icons and descriptions.
   - Select an option (e.g., "No tengo ideas").
   - Click "Siguiente".

## 3. Business Dashboard (Split View)
1. Verify the layout shows two main panels side-by-side.
   - **Left**: Chat Interface
   - **Right**: Offers Dashboard
2. **Header**:
   - Check the "Topic / Interest" (or "Business Idea / Problem") input field.
   - Verify it allows typing.
3. **Chat**:
   - Verify the initial message greets you based on your selected country/situation.
   - Type a message (e.g., "Me gusta el fitness").
   - Confirm the AI replies.
4. **Offers**:
   - Update the "Topic / Interest" field with a concrete idea (e.g., "Entrenador personal para ejecutivos").
   - Click **"Diseñar mis Ofertas High-Ticket"** (or "Design my offers").
   - Wait for the generation (loading state).
   - Verify 3 offers are displayed with prices, timeline, and details.
   - Click "Copy Outreach" to test clipboard functionality.

## 4. Manual Problem Input
1. Edit the "Topic / Interest" input field at the top of the chat column.
2. Change it to something different (e.g., "Consultoría de IA").
3. Click "Generate Offers" again.
4. Verify the new offers are relevant to the updated topic.
